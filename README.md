# Makeup Products Table

This project is a React application built as part of the Trainee React Developer screening for Grandia Solutions.

The application displays cosmetic products fetched from the Makeup API in a table format and provides grouping, filtering, and expandable rows functionality.

---

## Features

- Product table built with Ant Design
- Data fetched from the Makeup API
- Expandable rows to display available product colors
- Grouping products by:
  - Brand
  - Category
  - Product type
- Multi-select filters:
  - Brands
  - Tags
- Automatic data fetching on filter change
- Loading and error handling states
- Deployed on Vercel

---

## Tech Stack

- React
- TypeScript
- Ant Design
- Vite
- Makeup API
- Vercel
